# Radx Real Estate

A modern, responsive real estate website for **Radx Real Estate**, a multifamily real estate solutions company based in Addis Ababa, Ethiopia.

![Radx Real Estate Logo](assets/Radx%20logo.png)

## Overview

Radx Real Estate is a static website designed to showcase property listings, services, and company information. The site features a clean, professional design optimized for both desktop and mobile viewing.

## Features

- **Responsive Design**: Mobile-friendly navigation and layout
- **Property Listings**: Browse luxury apartments and office spaces
- **Service Information**: Learn about buying, selling, investment consulting, and property management
- **Company Profile**: About us, mission, vision, and team information
- **Contact Form**: Easy-to-use registration and contact forms
- **Social Media Integration**: Links to Facebook, X (Twitter), and LinkedIn

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- A local web server (optional, for development)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/radx-realstate.git
   cd radx-realstate
   ```

2. Open `index.html` in your browser or serve locally:
   ```bash
   # Using Python
   python -m http.server 8080
   
   # Using Node.js (http-server)
   npx http-server -p 8080
   ```

3. Visit `http://localhost:8080` in your browser
